# Vizzy - Criador de Infográficos

Um aplicativo web para criar infográficos interativos de forma fácil e rápida.

## Funcionalidades

- Seleção de tipos de gráfico (barras, linhas, pizza, progresso)
- Entrada de dados manual, via JSON ou CSV
- Visualização em tempo real
- Download de imagens dos gráficos

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Framer Motion

## Como executar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Rode o desenvolvimento: `npm run dev`
4. Abra http://localhost:8081
